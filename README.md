The main part of the programme is Stereo Matching and Depth Calculations, for stereo matching, I divided image as patches and match each patch's grayscale. I met some problems with this method. If there's a big pure-single-color area on the image, if the patch's size is set too low, it loses accuracy, if the patch's size set too high, it loses details.
